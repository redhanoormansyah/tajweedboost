# Memulai Perjalanan Tajwid Anda

Ahlan wa sahlan! Ini adalah contoh artikel blog pertama Anda di website **Tajweed Boost**.

Untuk membuat artikel baru, Anda cukup:
1. **Menduplikasi file ini** (`artikel-pertama.md`).
2. Mengganti nama file menjadi judul artikel Anda, misalnya `hukum-nun-mati.md`.
3. Mengubah bagian atas file ini (disebut *Frontmatter*) untuk mengatur `title` (Judul), `description` (Deskripsi singkat), dan `pubDate` (Tanggal terbit).

## Kenapa Menggunakan Markdown?

Markdown sangat mudah digunakan untuk menulis konten web tanpa perlu belajar kode HTML yang rumit. Anda bisa membuat teks menjadi tebal dengan `**teks**`, miring dengan `*teks*`, dan membuat daftar seperti di atas.
